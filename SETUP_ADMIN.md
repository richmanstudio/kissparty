# Как установить первого администратора

Если у вас еще нет ни одного администратора, используйте один из следующих способов:

## Способ 1: Через config.py (самый простой)

1. Откройте файл `config.py`
2. Найдите строку с `ADMIN_USERS`
3. Добавьте ваш `user_id` в список:

```python
ADMIN_USERS = [875068919, 728776547, ВАШ_USER_ID]
```

4. Перезапустите бота

**Как узнать свой user_id?**
- Напишите боту [@userinfobot](https://t.me/userinfobot) - он покажет ваш ID

## Способ 2: Через SQL запрос в базе данных

1. Откройте phpMyAdmin или другой инструмент для работы с MySQL
2. Выполните SQL запрос:

```sql
UPDATE users SET role = 'admin' WHERE user_id = ВАШ_USER_ID;
```

Или если пользователя еще нет в базе:

```sql
INSERT INTO users (user_id, username, first_name, role) 
VALUES (ВАШ_USER_ID, 'ваш_username', 'Ваше Имя', 'admin')
ON DUPLICATE KEY UPDATE role = 'admin';
```

## Способ 3: Через команду /setadmin (если уже есть админ)

Если у вас уже есть хотя бы один админ (из `config.ADMIN_USERS`), он может назначить других:

```
/setadmin <user_id>
```

## Проверка

После установки админа:
1. Напишите боту `/start`
2. В главном меню должна появиться кнопка "⚙️ Админ-панель"
3. Нажмите на неё для доступа к админ-панели

## Назначение промоутеров

Для назначения промоутера используйте SQL:

```sql
UPDATE users SET role = 'promoter' WHERE user_id = <USER_ID>;
```

Или через админ-панель (если функция добавлена в будущем).
