# 💋 KISS PARTY PAY BOT

> Telegram-бот для продажи билетов на вечеринку **KISS PARTY** с полной экосистемой: оплата, бонусы, рефералы, промокоды, QR-билеты и панель администратора.

---

## Возможности

### 🎟 Покупка билетов
| Тип | Описание | Лимит |
|-----|----------|-------|
| Обычный | Стандартный вход | 600 шт. |
| VIP | VIP-зона | 80 шт. |
| VIP Стоячий | VIP без места | 20 шт. |
| Парный | Для двоих | 50 шт. |

- Выбор количества от 1 до 10+
- Скидка **10%** при покупке от 5 билетов
- Генерация QR-кода для каждого билета

### 💳 Оплата
- Перевод на карту вручную
- Подтверждение/отклонение заявки администратором
- После одобрения — автоматическая выдача QR-билетов

### 🎁 Бонусная система
- Начисление **5%** от суммы каждой покупки
- Оплата бонусами до **50%** стоимости заказа
- История начислений и списаний

### 👥 Реферальная программа
- Личная реферальная ссылка для каждого пользователя
- **50 бонусов** рефереру и рефералу за первую покупку приглашённого
- Статистика: количество рефералов и сумма заработка

### 🏷 Промокоды
- Процентные и фиксированные скидки
- Ограничение по типам билетов
- Лимит использований и срок действия
- Минимальная сумма заказа

### 🛡 Поддержка
- Система тикетов поддержки прямо в боте
- Переписка пользователя с администратором
- Статусы: открыт / в работе / ожидает / закрыт
- Приоритеты: низкий / средний / высокий / срочный

### 💘 Dating
- Профили участников вечеринки
- Лайки и матчи

### 📢 Промоутеры
- Отдельная панель для промоутеров
- Статистика по приглашённым

## Схема базы данных

| Таблица | Назначение |
|---------|------------|
| `users` | Пользователи (роли: user / admin / moderator / promoter) |
| `tickets` | Купленные билеты с QR file_id |
| `payment_requests` | Заявки на ручную оплату |
| `bonuses` | Бонусные балансы |
| `bonus_logs` | История операций с бонусами |
| `referrals` | Реферальные коды и связи |
| `referral_bonus_logs` | История реферальных начислений |
| `promocodes` | Промокоды и условия |
| `price_settings` | Цены на билеты (редактируются из админки) |
| `promo_tickets` | Бесплатные промо-билеты |
| `support_tickets` | Тикеты поддержки |
| `support_messages` | Переписка внутри тикетов |
| `broadcasts` | Рассылки (текст / фото / видео / голос / документ) |
| `bot_settings` | Настройки бота (включить продажи, текст меню и др.) |
| `user_states` | FSM-состояния пользователей |

## Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка конфигурации

Откройте `config.py` и заполните:

```python
BOT_TOKEN = "ваш_токен_бота"

DB_CONFIG = {
    'host': 'localhost',
    'user': 'пользователь_бд',
    'password': 'пароль_бд',
    'database': 'название_бд',
}

ADMIN_USERS = [ваш_telegram_id]
PAYMENT_CARD_NUMBER = "номер_карты_для_оплаты"
```

### 3. Создание схемы БД

```bash
# Через MySQL CLI:
mysql -u root -p название_бд < database_schema.sql
mysql -u root -p название_бд < support_tickets_schema.sql
```

### 4. Создание временной папки

```bash
mkdir tmp
```

### 5. Запуск

```bash
python bot.py
```

## Панель администратора

Доступна по команде `/admin` внутри бота. Возможности:

- Просмотр и обработка заявок на оплату
- Управление ценами на билеты
- Управление промокодами
- Рассылка сообщений всем пользователям
- Настройки бота (включить/выключить продажи, изменить текст меню)
- Статистика: продажи, пользователи, билеты
- Работа с тикетами поддержки

## Сканер QR-билетов

Файл `qr_scanner.html` — веб-сканер для входа на мероприятие. Открывается на телефоне организатора, сканирует QR-коды гостей и подтверждает их действительность.

Защищён токеном (`QR_SCANNER_TOKEN`), настраивается через переменную окружения:

```bash
export QR_SCANNER_TOKEN=ваш_секретный_токен
```
